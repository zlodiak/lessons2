facade это паттерн, который упрощает функциональность сложной системы объектов. например я подключил сложную библиотеку со множеством функций. мне хотелось бы чтобы пльзователь несколькими действиями запустил сложный процесс.

например пользователь ныжимает всего лишь одну кнопку, но в результате в интернете находится кусок текста, из него вырезаются нецензурные слова, внизу текста добавляется подпись, затем текстразмещается в виде поста в фейсбуке. тут хорошо, что пользователь не знает насколько процесс сложен изнутри:

class Op1:
    def action(self):
        print('find text in web')

class Op2:
    def action(self):
        print('delete obscene words')

class Op3:
    def action(self):
        print('put a signature')

class Op4:
    def action(self):
        print('post in facebook')                        

class Facade:
    def __init__(self, op1, op2, op3, op4):
        op1.action()
        op2.action()
        op3.action()
        op4.action()

op1 = Op1()
op2 = Op2()
op3 = Op3()
op4 = Op4()
facade = Facade(op1, op2, op3, op4)


по сути в магазинах продавец выполняет роль фасада, он упрощает покупателю процес покупки вцелом.
