встроенные типы данных:

None
Логический, может принимать одно из двух значений — True (истина) или False (ложь).
Числа. могут быть целыми (1 и 2), с плавающей точкой (1.1 и 1.2) и даже комплексными.
Строки — последовательности символов Юникода, например, HTML-документ.

Списки — упорядоченные последовательности значений.
Кортежи — упорядоченные неизменяемые последовательности значений.
Множества — неупорядоченные наборы значений. неизменяемые множества(frozenset)
Словари — неупорядоченные наборы пар вида ключ-значение.

Байты - неизменяемый список байтов(чисел от 0 до 255)
массивы байтов - изменяемый список байтов

----------------
дополнительные типы данных из библиотеки collections:
Counter - вид словаря, который позволяет считать количество объектов
deque - двусторонняя очередь из итерируемого объекта. есть возможность установить максимальную длину.
defaultdict - обычный словарь с возможностью значений по умолчанию
OrderedDict - словарь, который помнит последовательность заполнения ключами
namedtuple - кортеж с именованными элементами

дополнительный тип данных из библиотеки array:
array - позволяет хранить список из данных одного типа
