соритровка вставками:

сложность O(n*n)

движение по n-1 элементам, начиная с второго.

второй сравнивается с первым. если он меньше, то они меняются местами.
третий сравнивается с вторым. второй с первым.
и т.д.

def insert_sort(A):
    for top in range(1, len(A)):
        k = top
        while k > 0 and A[k-1] > A[k]:
            A[k], A[k-1] = A[k-1], A[k]
            k -= 1

A = [5, 3, 4, 1, 2]
print(A)
insert_sort(A)
print(A)
