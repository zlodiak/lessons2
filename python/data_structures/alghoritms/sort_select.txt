сортировка выбором.

есть несортированный массив чисел. предпологается, что a[0] минимальный элемент, он сравнивается с каждым из оставшихся, по ходу ищется наименьший. в конце итерации найденный наименьший элемент удаляется из массива a и вставляется в массив b. это повторяется для каждого из оставшихся в массива a элементов. в итоге имеем в массиве b отсортированный массив.
